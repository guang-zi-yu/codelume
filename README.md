# codelume

codelume 是 MacOS 平台的动态壁纸软件， 该仓库是整个 codelume 项目。

使用下面的命令拉取 codelume 源码：

```bash
git clone --recursive https://github.com/guang-zi-yu/codelume.git
```

## 仓库介绍

仓库包含以下几个子仓库：

- [codelume-minio](https://github.com/guang-zi-yu/codelume-minio.git)
是 codelume 项目的 MinIO 服务端，用于存储动态壁纸及其他对象化数据。


## 编译运行

编译运行方法参考各个子仓库的 README.md 文件。
